# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2026-02-28

### Added
- Auto-generate Address/Label constants via Source Generator
- Generate Keys button integrated into the Addressables Groups window
- Auto-simplify addresses (file-name based)
- Auto-sanitize labels (C# identifier compatible)
- Settings window (individual toggles for Address/Label constants)
- First-install setup popup for existing entries
- Menu: Window > Asset Management > Addressables Extension
